# Battleship matrix

Code puzzle solution. Goal is to parse arrays of strings and determine how many ships of each size exist.

## Running it

Prerequisites:

- NodeJS v25 (supports running TS files "as is" without pre-compilation) + NPM
- Typescript v5 (supporting importing TS extensions)
- Respect lf line endings and don't auto-convert to crlf, tests will not work if you do..

```sh
npm i
npm run test
```
